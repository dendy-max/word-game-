使用 HTML + Tailwind CSS + 原生JavaScript 制作三关连贯的小学生英语单词闯关游戏，单文件HTML，纯前端，无外部图片与音频，全部图形用CSS绘制，支持部署到Netlify与GitHub Pages，手机电脑都能正常打开。

整体框架：
首页有开始按钮，一关通关自动解锁下一关，全程计分，全部通关弹出电子奖状。

关卡1：教室单词翻牌记忆配对（简化版，无炸弹，难度低）
词汇：window、blackboard、door、desk、chair、schoolbag、have a look
游戏规则：
1. 页面生成多组翻面卡片，卡片一面是教室物品简笔画，另一面是对应的英文单词；
2. 玩家每次翻开2张卡片；
3. 如果物品图片和英文单词互相匹配，卡片自动消除并加分，同时语音朗读单词；
4. 如果两张卡片不匹配，卡片自动翻回背面，没有扣分、没有惩罚，可以无限次重试；
5. 所有卡片全部配对消除后，自动进入第二关。

关卡2：水果飞船发射消消大作战
词汇：lemon、banana、apple、watermelon、pear、pineapple、peach、orange
游戏规则：
1. 屏幕上方不断下落带水果图案+英文单词的方块；
2. 底部有一艘飞船，先点击飞船选定单词，再点击上方相同单词方块，即可发射炮弹消除方块；
3. 设置倒计时，方块触顶则游戏失败，可以重新再来；
4. 在时间内清空所有水果方块，解锁第三关。

关卡3：池塘小动物捕鱼选词游戏
词汇：tadpole、frog、butterfly、goldfish、duck、tortoise
游戏规则：
1. 池塘内随机游动多条小鱼，鱼身上印有动物英文单词；
2. 浏览器自动朗读目标单词；
3. 学生拖动渔网，去捕捞带有对应单词的小鱼；
4. 捕捞正确加分，捕捞错误无惩罚；完成6道单词任务后，弹出总得分与通关奖状。

通用硬性要求：
1. 整体画风卡通童趣，色彩明亮，适合低年级小学生；
2. 全部单词支持Web Speech API英文朗读；
3. 每一关都可以随时重置重玩；
4. 代码结构清晰带注释，方便后续修改单词列表；
5. 无任何本地资源依赖，网页打开直接运行，不会出现404。

只输出完整的单文件HTML代码，不要多余文字。
